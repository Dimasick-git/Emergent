# Emergent Mobile App

Мобильное приложение для Emergent - платформы для командной коммуникации. Разработано на React Native с использованием Expo.

## Особенности

✨ **Полная поддержка чатов** - Отправляйте и получайте сообщения в реальном времени  
📱 **Нативная мобильная версия** - Работает на iOS и Android  
🔐 **Безопасная аутентификация** - JWT токены и защита данных  
🎯 **Управление каналами** - Создавайте и управляйте каналами  
👤 **Профиль пользователя** - Управляйте своим аккаунтом  

## Установка

### Требования

- Node.js 16+
- npm или yarn
- Expo CLI

### Шаги установки

```bash
# Перейти в папку мобильного приложения
cd mobile

# Установить зависимости
npm install

# Запустить приложение
npm start
```

## Доступные команды

```bash
# Запустить на мобильном эмуляторе iOS
npm run ios

# Запустить на Android эмуляторе
npm run android

# Запустить в браузере
npm run web

# Запустить тесты
npm test

# Проверить код
npm run lint

# Собрать приложение
npm run build
```

## Структура проекта

```
mobile/
├── src/
│   ├── app/                 # Экраны приложения (Expo Router)
│   │   ├── login.tsx        # Экран входа
│   │   ├── _layout.tsx      # Главный layout
│   │   └── (root)/          # Основные экраны приложения
│   │       ├── chat.tsx     # Экран чатов
│   │       ├── channels.tsx # Экран каналов
│   │       ├── profile.tsx  # Профиль пользователя
│   │       └── _layout.tsx  # Layout для табов
│   ├── api/                 # API клиенты
│   │   ├── auth.api.ts      # Аутентификация
│   │   ├── messages.api.ts  # Сообщения
│   │   └── channels.api.ts  # Каналы
│   └── store/               # Zustand stores
│       ├── auth.store.ts    # Управление аутентификацией
│       └── chat.store.ts    # Управление чатами
├── app.json                 # Конфиг Expo
├── package.json             # Зависимости
└── tsconfig.json            # TypeScript конфиг
```

## Конфигурация

Измените `API_URL` в `src/api/auth.api.ts` для подключения к вашему серверу:

```typescript
const API_URL = 'http://localhost:3001/api'; // для локальной разработки
```

## Экраны приложения

### Login Screen
- Вход в аккаунт
- JWT аутентификация
- Сохранение токена

### Chat Screen
- Список всех чатов
- Список последних сообщений
- Обновление в реальном времени

### Channels Screen
- Просмотр всех каналов
- Информация о канале
- Количество участников

### Profile Screen
- Информация пользователя
- Настройки приложения
- Выход из аккаунта

## API Интеграция

Приложение подключается к Emergent Backend API:

```
POST /api/auth/login          # Вход
GET  /api/messages/conversations # Получить чаты
GET  /api/messages/:id        # Получить сообщения
POST /api/messages            # Отправить сообщение
GET  /api/channels            # Получить каналы
```

## Общие проблемы

### Проблема: "Cannot find module"
**Решение:** Запустите `npm install` еще раз

### Проблема: Сервер не доступен
**Решение:** Убедитесь, что backend запущен на `http://localhost:3001`

### Проблема: Ошибка при подключении к API
**Решение:** Проверьте `API_URL` в `src/api/auth.api.ts`

## Разработка

### Добавление нового экрана

1. Создайте файл в `src/app/` или `src/app/(root)/`
2. Используйте стандартные React Native компоненты
3. Подключитесь к store через Zustand
4. Используйте API клиентов для запросов

### Добавление нового API

1. Создайте новый файл в `src/api/`
2. Используйте экземпляр `api` из `auth.api.ts`
3. Экспортируйте функции
