# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-04-06

### Added
- Initial project setup and structure
- Backend: NestJS framework with TypeScript
- Backend: PostgreSQL database with Prisma ORM
- Backend: WebSocket Gateway for real-time messaging
- Backend: JWT authentication with refresh tokens
- Backend: MinIO integration for file storage
- Backend: OpenAPI/Swagger documentation
- Frontend: React with Vite and TailwindCSS
- Frontend: WebSocket client for real-time updates
- Frontend: Zustand for state management
- Mobile: React Native setup with Expo
- SDK: TypeScript SDK for API integration
- SDK: Python SDK for API integration
- SDK: Go SDK for API integration
- CLI: Command-line utility for Emergent management
- Docker: docker-compose orchestration
- Docker: Multi-container environment
- CI/CD: GitHub Actions workflows (CI/CD/Pages/Docs)
- Documentation: Complete API documentation
- Documentation: Architecture overview
- Documentation: Database schema
- Tests: Unit tests for backend
- Tests: E2E tests setup
- Tests: Load testing configuration

### Features
- Workspace management
- Personal and group chats
- Markdown with code highlighting
- File sharing
- Voice messages
- Threading support
- Message reactions
- Real-time WebSocket communication
- Secure authentication and authorization
- User profiles and avatars

### Infrastructure
- PostgreSQL database
- Redis for caching and real-time
- MinIO for S3-compatible storage
- Nginx reverse proxy
- Docker containerization
- GitHub Pages deployment
- GitHub Container Registry
